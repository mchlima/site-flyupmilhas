<objective>
Site launches in a few days. Until then, visitors hitting the public homepage must see a branded pre-launch teaser — NOT the real landing. The teaser needs to feel mysterious and impactful (dark, minimal, on-brand) without giving away the offering.

Mechanism: a runtime-configurable feature flag (`NUXT_PUBLIC_COMING_SOON`) gates the homepage render. When flipped off, the full landing returns instantly with no code change.

Output:
1. New `app/components/ComingSoon.vue` — full-viewport dark teaser with logo, headline, subtle motion
2. Modified `app/pages/index.vue` — conditionally renders `<ComingSoon />` or the current landing
3. Modified `nuxt.config.ts` — adds `comingSoon: true` to `runtimeConfig.public`

Non-goals: admin routes (`/admin`, `/pagamento`, `/encontros`) remain accessible. No countdown timer (user said "alguns dias" — no exact date given, and countdowns require commitment). No email capture form (adds complexity + backend coupling for a temporary page). No analytics wiring on teaser (site-level GA already runs; no page-level events needed).

**Step 3 — Modify `app/pages/index.vue`:** Gate the full landing behind the flag. Keep existing SEO/schema for the real site (unchanged), but when teaser is active, swap to minimal meta + noindex.

Replace the entire current `app/pages/index.vue` with:

```vue
<script setup lang="ts">
definePageMeta({ layout: false })

const config = useRuntimeConfig()
const isComingSoon = computed(() => Boolean(config.public.comingSoon))

// Coming-soon: minimal meta, noindex. Real landing: full SEO stack.
if (isComingSoon.value) {
  useSeoMeta({
    title: 'Fly Up Milhas — Em breve',
    description: 'Algo novo está decolando. Volte em breve.',
    robots: 'noindex, nofollow',
    ogTitle: 'Fly Up Milhas — Em breve',
    ogDescription: 'Algo novo está decolando. Volte em breve.',
  })
}
else {
  useSeoMeta({
    title: 'Fly Up Milhas — Mentoria em Milhas Aéreas',
    description: 'Mentoria para aprender a acumular milhas e voar em classe executiva gastando menos.',
    ogTitle: 'Fly Up Milhas — Mentoria em Milhas Aéreas',
    ogDescription: 'Mentoria para aprender a acumular milhas e voar em classe executiva gastando menos.',
    ogImage: 'https://cdn.flyupmilhas.com.br/og-image.jpg',
    ogImageWidth: 1200,
    ogImageHeight: 630,
    ogType: 'website',
    ogUrl: 'https://flyupmilhas.com.br',
    ogLocale: 'pt_BR',
    twitterCard: 'summary_large_image',
    twitterTitle: 'Fly Up Milhas — Mentoria em Milhas Aéreas',
    twitterDescription: 'Mentoria para aprender a acumular milhas e voar em classe executiva.',
    twitterImage: 'https://cdn.flyupmilhas.com.br/og-image.jpg',
  })

  useSchemaOrg([
    defineLocalBusiness({
      '@type': 'ProfessionalService',
      name: 'Fly Up Milhas',
      description: 'Mentoria em acúmulo e emissão estratégica de milhas aéreas',
      url: 'https://flyupmilhas.com.br',
      areaServed: {
        '@type': 'Country',
        name: 'Brasil',
      },
      knowsAbout: ['milhas aéreas', 'passagens executivas', 'programas de fidelidade'],
    }),
  ])
}
</script>

<template>
  <ComingSoon v-if="isComingSoon" />
  <div v-else>
    <AppHeader />
    <main id="main-content" class="pt-16">
      <SectionHero />
      <SectionMarketTicker />
      <SectionAbout />
      <SectionProgramContent />
      <SectionForWhom />
      <SectionMethod />
      <SectionProvas />
      <SectionSocialProof />
      <SectionPrice />
      <SectionGuarantee />
      <SectionFAQ />
      <SectionFinalCTA />
      <SectionLeadForm />
    </main>
    <AppFooter />
    <BackToTop />
  </div>
</template>
```
