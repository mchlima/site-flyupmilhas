**Create app/components/BackToTop.vue (per D-17 through D-21, UX-01):**

Create a new component with the following implementation:

```vue
<script setup lang="ts">
const isVisible = ref(false)

function scrollToTop() {
  window.scrollTo({ top: 0, behavior: 'smooth' })
}

if (import.meta.client) {
  const SCROLL_THRESHOLD = 300

  onMounted(() => {
    const onScroll = () => {
      isVisible.value = window.scrollY > SCROLL_THRESHOLD
    }
    window.addEventListener('scroll', onScroll, { passive: true })
    onUnmounted(() => {
      window.removeEventListener('scroll', onScroll)
    })
  })
}
</script>
```

Template:
- A `<Transition>` wrapper with fade animation (name="fade")
- Inside: a `<button>` with `v-show="isVisible"` (per D-17: bottom-right corner, appears after ~300px scroll)

- `aria-label="Voltar ao topo"` for accessibility
- Inside button: `<UIcon name="i-heroicons-chevron-up" class="w-6 h-6" />` (per D-19)
