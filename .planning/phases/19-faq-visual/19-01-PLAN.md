  <action>
Rewrite the template and styles in SectionFAQ.vue to implement the numbered card design. Keep the existing faqItems array and script setup unchanged.

**Card wrapper (per D-01, D-08):**
- Wrap the UAccordion in a container with `space-y-3` for gaps between cards (per D-04, ~12px gap)
- Each accordion item rendered inside UAccordion slots gets card styling via the :ui prop and wrapper markup

**UAccordion :ui prop (per D-14, CRITICAL for FAQ-03):**
Replace the current :ui prop with a new one that includes:
```
{
  trigger: 'group flex-1 flex items-center gap-3 font-semibold text-base py-4 px-5 bg-white rounded-xl border border-gray-100 min-w-0 text-[var(--color-brand-text)] focus-visible:ring-2 focus-visible:ring-offset-2 focus-visible:ring-[var(--color-brand-primary)]',
  content: 'bg-white rounded-b-xl border-x border-b border-gray-100 -mt-3 pt-0'
}
```
IMPORTANT: The trigger class string MUST include `focus-visible:ring-2 focus-visible:ring-offset-2 focus-visible:ring-[var(--color-brand-primary)]` because the :ui prop REPLACES (not merges) default Nuxt UI classes, silently removing focus indicators if omitted.

**Number display (per D-02, D-09):**
In the #default slot, use the `index` parameter from the slot scope. Display the number as:
```html
<span class="text-lg font-bold text-[var(--color-brand-primary)] tabular-nums shrink-0">
  {{ String(index + 1).padStart(2, '0') }}
</span>
```

**Question text (per D-05, FAQ-02):**
The question label in the #default slot should use:
```html
<span class="text-[var(--color-brand-text)] font-semibold text-base text-left">{{ item.label }}</span>
```

**Chevron indicator (per D-03):**
Add a chevron icon on the right side of the trigger. Use UAccordion's built-in trailing icon if available, or add a manual SVG chevron that rotates when open:
```html
<UIcon name="i-lucide-chevron-down" class="shrink-0 size-5 text-[var(--color-brand-text-muted)] transition-transform duration-200 group-data-[state=open]:rotate-180" />
```

**Answer text (per D-06, FAQ-02):**
In the #content slot:
```html
<p class="text-[var(--color-brand-text-muted)] text-sm leading-relaxed pb-4 px-5 pl-14">
  {{ item.content }}
</p>
```
The `pl-14` aligns the answer text with the question text (past the number column).

**Section background (per D-07):**
Keep existing `bg-[var(--color-brand-bg)]` on the section element.

**Accordion behavior (per D-10, D-11):**
Keep single-open behavior (UAccordion default). Keep default transition.

**Discretion choices:**
- Number size: text-lg font-bold (prominent but not oversized)
- Cards use border (border-gray-100) rather than shadow for a cleaner look
- Card padding: py-4 px-5 on trigger
- Open card: no additional highlight (the expanded content is distinction enough)
