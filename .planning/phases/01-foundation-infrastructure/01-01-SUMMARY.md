## Decisions Made
- `tailwindcss` installed as direct dependency: although the plan says "@nuxt/ui manages Tailwind as peer dep", the `@import "tailwindcss"` directive in main.css requires the package to be resolvable in node_modules. This is the correct Tailwind v4 behavior — the peer dep still needs to be present.
- `@takumi-rs/wasm@^1.0.0-beta.3` installed: `@nuxtjs/seo` includes `nuxt-og-image` which requires this WASM OG image renderer. Without it, the Nitro build fails with a missing module error.
- `zod` pinned to v3 (installed v3.25.76): npm defaulted to installing zod v4 (4.3.6); corrected to v3 per research requirement.
