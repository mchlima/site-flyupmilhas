## 1. Visão Geral do Produto
O objetivo é criar uma Landing Page (LP) de alta conversão para a **Fly Up Milhas**. O foco central é a venda de consultoria personalizada para pessoas que desejam otimizar o acúmulo de milhas e realizar emissões de passagens aéreas de forma estratégica.

### 🎯 Objetivos de Negócio
1.  **Conversão Direta:** Transformar visitantes em clientes da consultoria.
2.  **Autoridade:** Posicionar o especialista como referência no mercado de milhas.
3.  **Captação de Leads:** Coletar dados de potenciais clientes para futuras ofertas.

## 3. Estrutura da Landing Page (Sessões)

### 3.1 Hero Section (Impacto)
* **Visual:** Elegante, fundo claro (Off-white), imagem de alta qualidade (estilo "lifestyle" ou aviação).
* **Copy:** "Sua próxima viagem de classe executiva custa menos que um jantar fora."
* **CTA:** Botão "Quero minha Consultoria".

### 3.2 O Especialista (Autoridade)
* Breve biografia do Márcio focada em resultados reais: milhas acumuladas, países visitados e economia gerada para alunos/clientes.

### 3.3 Como Funciona (O Método)
Apresentação visual em 3 ou 4 passos:
1.  **Diagnóstico:** Análise do perfil de gastos e cartões atuais.
2.  **Estratégia:** Definição dos melhores programas para o seu objetivo.
3.  **Execução:** Reunião guiada para realizar as transferências e emissões.
4.  **Voo:** Suporte durante o processo de check-in e viagem.

### 3.4 Prova Social
* Espaço dedicado a prints de resultados e depoimentos de clientes satisfeitos (foco em economia financeira).

### 3.5 Formulário de Qualificação (Leads)
Campos sugeridos para filtrar clientes ideais:
* Nome e WhatsApp.
* Média de gastos mensais no cartão de crédito.
* Principal objetivo (Viagem de luxo, economia em família, renda extra).

## 4. Requisitos de Design & UX

* **Paleta de Cores:**
    * **Primária:** Azul Aviação (Confiança e Professionalismo).
    * **Fundo:** Branco Gelo ou Cinza Ultra Claro (Clean e Moderno).

* **Performance:** Carregamento ultra rápido (Mobile First) via Cloudflare R2.

## 5. Requisitos Técnicos (Stack)

* **Frontend:** Nuxt 3 (SSR para performance e SEO).
* **UI Framework:** Nuxt UI (Tailwind CSS + Headless UI).
* **Backend:** Fastify API (Já configurado).

* **Storage:** Cloudflare R2 (Imagens e ativos estáticos).
